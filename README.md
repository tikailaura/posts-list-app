# posts-list-app

Delivered a dynamic frontend application that loads API data in paginated batches, rendering post content into a structured and interactive interface. Added debounced search and keyword highlighting to improve usability while avoiding unnecessary requests — demonstrating practical experience with data-driven UI, list rendering, and performance-conscious frontend behavior.

Tech Stack :

- HTML5
- CSS3
- JavaScript
- Fetch API
- JSONPlaceholder API
